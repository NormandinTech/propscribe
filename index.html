How compliance works

Protection built into every word

The Fair Housing Act carries real penalties. PropScribe is the only marketing tool that treats compliance as a first-class feature, not an afterthought.

1
You enter property details
Address, beds, baths, features, price. No demographic information — PropScribe doesn't need it and doesn't want it.
2
AI writes the content
A specialized real estate writing model generates copy using only objective property attributes. Protected class language is blocked at the model level, not filtered after.
3
Compliance audit runs automatically
A second AI pass audits the output against 9 HUD violation categories. Each flag includes the exact phrase, the legal concern, and a compliant alternative.
4
Auto-rewrite if score is below 85
If any compliance issue is found, PropScribe automatically regenerates with the flagged language corrected. You only see output that's ready to publish.
